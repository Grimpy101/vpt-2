# VPT 2.0 API

## Development
The project requires
* ``node`` (either 12.22.0, 14.17.0, or at least 16.0.0)
* ``yarn`` package manager.

For first time setup run
```
yarn
yarn install
```

To compile and run the node server:
```
yarn build
```

To just run the node server:
```
yarn start
```

To run ESLint
```
yarn lint
```
